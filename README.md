# AI Resume Analyzer

A web-based Resume Analyzer built using Python and Streamlit.

## Features

* Upload Resume PDF
* Extract Resume Text
* Detect Technical Skills
* Calculate Resume Score
* Identify Missing Skills
* Visualize Results with Bar Charts

## Technologies Used

* Python
* Streamlit
* PyPDF2
* Matplotlib

## How It Works

1. Upload a PDF resume.
2. Extract text from the resume.
3. Detect technical skills.
4. Calculate a resume score.
5. Display missing skills.
6. Generate a skills analysis chart.

## Installation

pip install -r requirements.txt

streamlit run app.py
